# inStock - Inventory Management System

Description
inStock is a desktop-based software solution designed to facilitate efficient warehouse inventory management. Developed using Python and the CustomTkinter framework, the application features a modern dark-themed user interface and utilizes SQLite for reliable local data storage.

Key Features
The application includes the following core functionalities:

- Inventory Data Management (CRUD): Allows users to create, read, update, and delete inventory items.
- Modern User Interface: Features a responsive dark mode design for enhanced user experience.
- Real-time Information: Displays current day, date, and time information on the main dashboard.
- Reporting and Analytics: Provides statistical summaries including total item types, physical stock counts, and total asset value.
- Data Export: Includes functionality to export inventory reports to CSV file format for further administrative processing.

Technologies Used
- Python 3
- CustomTkinter (GUI Framework)
- SQLite3 (Database)
- CSV (Data Export)

Installation and Usage
Follow these steps to run the application on a local machine:

1. Clone or download this repository to your computer.
2. Ensure Python 3 is installed on your system.
3. Install the required dependencies by running the following command in the terminal:
   pip install customtkinter
4. Run the application using the command:
   python app.py
